# How to run Project

## install node
See [NodeJS](https://nodejs.org/en/download/current/).

## install mongodb
See [MongoDB](https://www.mongodb.com/try/download/community).

## install vue-CLI
```
npm install -g @vue/cli
```

## frontend setup
```
npm i
```

## backend setup
```
npm i --force
```

## uninstall package
```
npm uninstall 
```

## start project
```
npm run dev
```

## Vue Cli
See [Vue Cli](https://cli.vuejs.org/config/).
